## ✨ Description

```
피부에 고민이 많은 20대 여성들을 대상으로한 화장품 쇼핑몰 제작 프로젝트입니다.
```

## 📌 Project Goal

```
 **목적 | 지금까지 배운 내용들을 학습하고 적용하기**

강의에서 배운 내용들을 실제로 적용해보면서 체득하고 오류를 해결과는 과정에서 내가 무엇을 모르는지 깨닫는 것이 목적입니다!

다른 팀원들과 협업하며 소통능력과 협업 툴에 익숙해지는 것 또한 부가적인 목표입니다.
```
